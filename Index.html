Настройка

Способ 1. Без .htaccess

Укажите путь к вашим файлам

style.less

style.scss

style.coffee

скопировано
  • # Less
  • <link rel="stylesheet" type="text/css"
    {{less.beforePath}}{{less.path}}{{less.afterPath}}
  • # scss
  • <link rel="stylesheet" type="text/css"
    {{scss.beforePath}} {{scss.path}}{{scss.afterPath}}
  • # coffee
  • <script type="text/javascript"
    {{coffee.beforePath}}{{coffee.path}}{{coffee.afterPath}}</script>

02

Способ 2. с .htaccess

В .htaccess после: RewriteEngine On

скопировано
  • # Less
  • RewriteRule ^(.*).less.css$
    https://compilim.ru/v2/c/less/?from_file=%{HTTP_HOST}/$1.less&ssl=%{HTTPS} [L]
  • RewriteRule ^(.*).less.min.css$
    https://compilim.ru/v2/c/less/?from_file=%{HTTP_HOST}/$1.less&ssl=%{HTTPS}&min=on [L]
  • # Scss
  • RewriteRule ^(.*).scss.js$
    https://compilim.ru/v2/c/scss/?from_file=%{HTTP_HOST}/$1.scss&ssl=%{HTTPS} [L]
  • RewriteRule ^(.*).scss.min.js$
    https://compilim.ru/v2/c/scss/?from_file=%{HTTP_HOST}/$1.scss&ssl=%{HTTPS}&min=on [L]
  • # Coffee
  • RewriteRule ^(.*).coffee.js$
    https://compilim.ru/v2/c/coffee/?from_file=%{HTTP_HOST}/$1.coffee&ssl=%{HTTPS} [L]
  • RewriteRule ^(.*).coffee.min.js$
    https://compilim.ru/v2/c/coffee/?from_file=%{HTTP_HOST}/$1.coffee&ssl=%{HTTPS}&min=on [L]

В шаблон добавьте ссылку на ваш файл

скопировано
  • # Less
  • <link rel="stylesheet" type="text/css" href="/css/style.less.css" />
  • <link rel="stylesheet" type="text/css" href="/css/style.less.min.css" />
  • # Scss
  • <link rel="stylesheet" type="text/css" href="/css/style.scss.css" />
  • <link rel="stylesheet" type="text/css" href="/css/style.scss.min.css" />
  • # Coffee
  • <script type="text/javascript" src="/js/script.coffee.js"></script>
  • <script type="text/javascript" src="/js/script.coffee.min.js"></script>
